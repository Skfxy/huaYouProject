# v0.1.0 MVP 功能补全与发布

- **日期**: 2026-07-15
- **版本**: v0.1.0
- **类型**: ✨ 功能优化
- **状态**: - [x] 已完成

## 更新内容

### 1. 时间推进系统增强
- [x] `packages/shared/src/types/game.types.ts` — TimeState 新增 `dayOfWeek`（星期几）、`consecutiveRegularSleep`（连续规律作息天数）、`stayedUpLate`（熬夜标记）；EventCondition 新增 `days`、`excludeFlags`；EventOption 新增 `timeAdvance`、`skipPeriods`
- [x] `packages/shared/src/constants/game.enum.ts` — 新增 DAYS_OF_WEEK、WEEKEND_DAYS、SPECIAL_DATES 常量；新增 AUTO_SAVE_CONFIG 自动存档配置
- [x] `packages/shared/src/config/balance.config.ts` — 新增 SLEEP_RECOVERY（睡眠恢复配置，含熬夜惩罚/周末加成/规律作息加成）、WEEKEND_CONFIG、PAYDAY_CONFIG、MONTH_END_CONFIG
- [x] `packages/shared/src/utils/calculator.ts` — 新增 `getDayOfWeek()`、`isWeekend()`、`isPayday()`、`isMonthEnd()`、`calculateSleepRecovery()`、`getSkillMaxLevel()`；修复技能升级使用各技能独立 maxLevel
- [x] `packages/web/src/stores/game.store.ts` — TimeState 跟踪 dayOfWeek/consecutiveRegularSleep/stayedUpLate；新增 `recordRegularSleep()`、`recordOvertimeSleep()`、`goToSleep()`；修正 isPayday 计算含第1天
- [x] `packages/web/src/data/events/date-events.json` — 新增6个特殊日期事件：周六早晨/周六晚间/周日早晨/发薪日/月末总结/首个周末

### 2. 睡眠恢复机制
- [x] 正常睡眠恢复精力35 + 心情2 + 健康1
- [x] 熬夜惩罚：精力恢复不足(-15)、健康-2、心情-5
- [x] 周末懒觉额外恢复：精力+15、心情+2
- [x] 连续7天规律作息：恢复系数×1.3、心情+3、健康额外+3
- [x] `packages/web/src/stores/player.store.ts` — 新增 `sleep()`、`applyOvertimePenalty()`、`sleepAfterOvertime()`、`applyPayday()`、`applyMonthlySummary()`

### 3. 事件引擎整合
- [x] `packages/web/src/hooks/useEventEngine.ts` — 整合睡眠机制、特殊日期事件、自动存档触发、事件优先级系统；新增 `handleDayChange()` 处理日期变更（睡眠/熬夜/发薪日/月末总结）；修复周末奖励重复发放和熬夜惩罚数值；新增特殊日期提示和自动存档触发
- [x] `packages/web/src/data/events/daily-events.json` — 新增日常自由活动事件（学习/工作/休息/社交/睡觉），每个时段有不同选项

### 4. 自动存档机制
- [x] `packages/web/src/stores/save.store.ts` — 实现 autoSave() 功能，localStorage 持久化；新增自动存档提示（showAutoSaveHint，1.5秒后自动隐藏）；关键节点触发：每5个选项、每天早晨、章节结束

### 5. UI 界面补全
- [x] `packages/web/src/components/common/BaseModal.vue` — 新建通用弹窗组件（Teleport + Transition，支持标题/关闭）
- [x] `packages/web/src/components/game/StatusDetailModal.vue` — 新建状态详情弹窗（四维状态描述、综合效率、效率文本、小贴士）
- [x] `packages/web/src/components/game/SkillPanelModal.vue` — 新建技能面板弹窗（技能总等级、各技能进度条/经验条/升级提示、技能说明）
- [x] `packages/web/src/components/game/StatusBar.vue` — 新增星期显示、周末标识；状态栏/技能面板可点击打开详情弹窗
- [x] `packages/web/src/views/GameView.vue` — 新增特殊日期提示、自动存档提示、星期/时段显示；结局界面显示评分和等级

### 6. 数值平衡调整
- [x] 正常睡眠恢复 40→35，熬夜补觉 20→15，周末额外 15→10，规律作息系数 1.25→1.3
