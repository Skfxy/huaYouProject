# 角色档案

> 《话游》角色设定文档

## 主角

### 基础信息

| 属性 | 内容 |
|------|------|
| 姓名 | 玩家自定义（默认：小明） |
| 年龄 | 22岁（2018年） |
| 专业 | 机械工程 |
| 学历 | 本科 |
| 毕业院校 | 普通一本（可自定义） |

### 原时间线经历

- 毕业后按部就班进入工厂，从事本专业工作
- 月薪从4500逐步涨到8000，但始终感觉被困住
- 看着同学转行互联网后薪资翻倍，内心充满悔恨
- 35岁时遭遇职业瓶颈，后悔当初没有转行

### 穿越后状态

- **记忆保留**：记得原时间线的所有经历，包括对未来的"预知"
- **心态变化**：从迷茫转为坚定，知道自己想要什么
- **核心目标**：抓住2018年互联网红利，改变人生轨迹
- **性格特点**：外向但社恐，执行力强但容易焦虑

### 起始数值

```
精力：80
心情：30
健康：70
存款：3200元
```

## 配角

### 小张（室友）

**基本信息**：
- 主角的大学室友，关系不错
- 专业：机械工程
- 性格：随和、有点佛系、喜欢游戏

**剧情作用**：
- 新手引导NPC，帮助主角确认穿越
- 随机事件触发者（开黑邀请等）
- 后续可发展为好友系统中的角色

**好感度系统**：
- 初始好感度：50
- 影响因素：是否一起开黑、是否分享信息、是否帮助他
- 高好感度解锁：小张的内推机会、特殊剧情

**后续发展**：
- 选择运营路线：小张可能成为同事
- 选择程序员路线：小张可能成为创业伙伴
- 延迟选择路线：小张可能成为对比对象

---

### 学长（图书馆偶遇）

**基本信息**：
- 已毕业一年的学长
- 目前在某大厂做前端开发
- 性格：热心、愿意帮助后辈

**剧情作用**：
- 转行路线的引路人
- 提供面试信息和学习建议
- 解锁「学长联系方式」道具

**触发条件**：
- 白天去图书馆
- 已选择转行路线

**后续发展**：
- 可能提供内推机会
- 分享面试经验
- 成为人脉系统的一部分

---

### HR/面试官（后续章节）

**首次出现**：第二章

**剧情作用**：
- 面试系统的核心NPC
- 影响主角的求职结果
- 不同路线遇到不同类型的HR

---

### 同事（后续章节）

**首次出现**：第二章/第三章

**剧情作用**：
- 职场社交系统的核心
- 影响主角的职场发展
- 可能成为好友或竞争对手

## 角色发展路线

### 第一章角色

| 角色 | 出场场景 | 作用 |
|------|----------|------|
| 主角 | 全程 | 玩家控制 |
| 小张 | 场景1-1, 1-2 | 引导、随机事件 |
| 学长 | 随机事件2 | 转行引导 |

| 角色 | 出场场景 | 作用 |
|------|----------|------|
| HR | 面试场景 | 求职判定 |
| 面试官 | 面试场景 | 技能判定 |
| 同事 | 入职场景 | 职场社交 |

### 第三章新增角色

| 角色 | 出场场景 | 作用 |
|------|----------|------|
| 领导 | 工作场景 | 晋升判定 |
| 同学 | 社交场景 | 对比参照 |
| 家人 | 回家场景 | 情感支持 |

## 角色语音风格

### 主角内心独白

- 语气：自嘲、吐槽、偶尔鸡汤
- 示例："凭什么啊……当年我要是也学了计算机，哪会有今天？"
- 示例："这次一定要抓住机会，不能再混下去了！"

### 小张

- 语气：随和、游戏宅、偶尔毒舌
- 示例："大清早的你发什么疯？答辩都过了，不多睡会儿？"
- 示例："来一把？反正也没事干。"

### 学长

- 语气：热心、经验分享、偶尔凡尔赛
- 示例："前端入门不难，关键是坚持。我当初也是零基础学的。"
- 示例："我们组最近在招人，你要不要试试？"
