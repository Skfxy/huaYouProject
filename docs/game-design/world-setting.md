# 世界观设定

> 《话游》的世界观核心：「一话一人生，一言一世界」

---

## 核心概念

### 平行时空

- 世界上存在无数条平行时间线，每一条都对应着人生的不同选择
- 主角的意识可以穿越到平行时空中，替换成那个时间线的"自己"
- 穿越并非回到过去，而是跳转到另一条时间线上"同名同姓"的自己身上

### 穿越触发

- 穿越发生在意识模糊的临界状态（如极度疲惫、即将入睡）
- 触发条件：强烈的悔恨/遗憾情绪 + 特定的"时空裂缝"时机
- 穿越后，原时间线的记忆保留，但无法主动返回

### 穿越限制

- 无法预知平行时空的完整走向
- 选择带来蝴蝶效应，后续发展不可完全掌控
- 每次"重来"都是一条全新的时间线，不存在读档重来

**宏观环境**：
- 互联网红利期顶峰，移动互联网高速发展
- 字节跳动疯狂招人，拼多多刚成立三年
- 小程序生态刚刚兴起，程序员薪资持续上涨

**生活成本**：
- 一线城市合租房：1500-2500元/月

- 餐饮：食堂套餐5-10元，外卖15-30元

**关键时间节点**：
- 6月：毕业季，校招收尾
- 7-8月：入职季，秋招准备期
- 9-11月：秋招高峰期
- 12月：年终，跳槽季前奏

## 主线分支

根据第一章的核心选择，游戏分为四条主线：

### 程序员路线

- **起点**：立刻转行学编程
- **挑战**：技术学习曲线陡峭，面试竞争激烈
- **机遇**：互联网高薪，职业天花板高
- **结局方向**：大厂offer / 创业成功 / 技术专家

### 稳扎稳打路线

- **起点**：先去工厂上班，业余学编程
- **挑战**：时间精力有限，容易半途而废
- **机遇**：有稳定收入，风险较低
- **结局方向**：成功转行 / 双面人生 / 回归本行

### 运营产品路线

- **起点**：不写代码，走运营/产品方向
- **挑战**：门槛低但竞争大，需要软实力
- **机遇**：沟通能力强可快速晋升
- **结局方向**：产品总监 / 运营专家 / 创业

### 延迟选择路线

- **起点**：先享受大学时光，再说
- **挑战**：错过最佳准备期，后期压力大
- **机遇**：人生不只有工作，体验也很重要
- **结局方向**：普通生活 / 后知后觉 / 意外机遇

## 世界观规则

### 蝴蝶效应

- 每个选择都会影响后续事件的触发条件
- 小选择可能带来大变化（如偶遇学长→获得内推机会）
- 无法完全预测选择后果，增加游戏真实感

### 时间不可逆

- 一旦做出选择，无法撤回
- 可以通过新周目体验不同选择
- 每周目都是独立的时间线

### 记忆保留

- 主角保留原时间线的记忆
- 这是优势（知道未来趋势）也是负担（知道哪些选择更好）
- 记忆不会随时间消退，但细节可能模糊
