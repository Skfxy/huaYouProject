# 第一章随机事件池

> 本章包含5个随机事件，用于丰富游戏体验

---

## 事件1：室友邀请开黑

### 基本信息

| 属性 | 内容 |
|------|------|
| 事件ID | event_001 |
| 事件名称 | 室友邀请开黑 |
| 事件类型 | 社交/娱乐 |
| 优先级 | 普通 |

### 触发条件

```
时间条件：晚上时段（18:00-24:00）
地点条件：宿舍
状态条件：精力 > 40
概率：每次晚上在宿舍时，30%概率触发
```

### 事件描述

**【场景】**
你正在宿舍里刷手机，小张从对面床铺探过头来。

**【小张】**
"来一把？反正也没事干。"

**【旁白】**
他的眼睛里闪着期待的光芒，显然已经等你很久了。

### 选项设计

**选项A：开黑放松一下**

| 属性 | 变化 |
|------|------|
| 心情 | +12 |
| 精力 | -8 |
| 时间 | 推进3小时 |
| 小张好感 | +3 |

**选项B：不了，我还有事**

| 属性 | 变化 |
|------|------|
| 心情 | -3 |
| 小张好感 | -2 |

### 后续影响

- 选择A：增加与小张的互动机会，解锁更多社交事件
- 选择B：减少社交机会，但有更多时间学习

| 事件名称 | 图书馆偶遇学长 |
| 事件类型 | 社交/学习 |

### 触发条件

```
时间条件：白天时段（6:00-18:00）
地点条件：图书馆

|------|------|

| 获得道具 | 学长联系方式 |
| 社交能力 | +1 |

**选项B：默默看书，不好意思打扰**

| 属性 | 变化 |
|------|------|
| 无变化 | - |

### 后续影响

- 选择A：获得「学长联系方式」道具，后续可触发内推事件
- 选择B：错过人脉积累机会

### 基本信息

| 属性 | 内容 |
|------|------|
| 事件ID | event_003 |
| 事件名称 | 银行卡余额提醒 |
| 事件类型 | 压力/经济 |
| 优先级 | 高 |

### 触发条件

```
时间条件：任意时段
地点条件：任意地点
状态条件：存款 < 2000

你愣了一下，毕业前还得交住宿费、聚餐费、可能还要买正装面试……钱好像不太够用了。

**选项A：省吃俭用**

| 属性 | 变化 |
|------|------|
| 心情 | -5 |
| 健康 | -2 |
| 存款 | 无变化 |

**选项B：找家里再要点**

| 属性 | 变化 |
|------|------|
| 存款 | +1500 |
| 心情 | -3（愧疚感） |

### 后续影响

- 选择A：短期节省，但影响健康和心情
- 选择B：解决燃眉之急，但有心理负担

## 事件权重系统

### 事件触发优先级

| 优先级 | 权重 | 说明 |
|--------|------|------|
| 高 | 0.4 | 必须触发或高概率触发 |
| 普通 | 0.3 | 正常触发概率 |
| 低 | 0.2 | 低概率触发 |
| 隐藏 | 0.1 | 需要特定条件触发 |

### 事件冷却机制

- 同一事件触发后，至少间隔2天才能再次触发
- 高优先级事件冷却时间为1天
- 隐藏事件无冷却时间，但触发条件严格

### 事件互斥机制

- 事件1（开黑）和事件5（面试）互斥
- 事件3（余额）和事件4（emo）可能同时触发
- 同一时段最多触发1个事件
