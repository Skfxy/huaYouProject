# 第三章：职场新人（规划中）

> 本章定位：职场成长体验，引入晋升系统、职场人际关系、长期目标
> 预计时长：90天（游戏内时间）
> 解锁内容：晋升系统、职场社交、长期目标系统

## 章节概述

### 时间线

- 2019年1月-3月
- 入职后的第一个季度

### 主线分支

根据第二章的选择，进入不同的职场路径：

| 主线 | 起点 | 核心挑战 |
|------|------|----------|

## 剧情大纲

### 第一幕：入职（第1-15天）

**场景 3-1：第一天**
- 入职报到，认识新同事
- 了解公司文化、工作流程
- 第一天的工作体验

**场景 3-2：培训期**
- 学习公司业务
- 与导师/师傅互动
- 第一个小任务

**场景 3-3：初步适应**
- 开始独立工作
- 遇到第一个困难
- 寻求帮助或独自解决

### 第二幕：成长（第16-45天）

**场景 3-4：第一个项目**
- 参与团队项目
- 发挥自己的技能
- 获得第一次认可

**场景 3-5：职场社交**
- 认识更多同事
- 参与团建活动
- 建立职场人脉

**场景 3-6：技能提升**
- 工作中学习新技能
- 业余时间自我提升
- 技能等级提升

### 第三幕：挑战（第46-75天）

**场景 3-7：第一个挑战**
- 重要项目/任务
- 压力与挑战
- 成长机会

**场景 3-8：职场政治**
- 同事关系处理
- 领导风格适应
- 职场生存法则

**场景 3-9：晋升机会**
- 晋升评估
- 准备晋升材料
- 晋升答辩/面试

### 第四幕：季度总结（第76-90天）

**场景 3-10：季度评估**
- 工作成果回顾
- 个人成长总结
- 下一季度目标

**场景 3-11：年终奖/转正**
- 转正评估
- 年终奖发放
- 职业规划调整

**场景 3-12：第三章结局**
- 章节总结
- 未来展望
- 解锁第四章

## 新增系统

### 晋升系统

**晋升等级**：
```
初级 → 中级 → 高级 → 专家 → 总监
```

**晋升条件**：
- 工作时长（经验积累）
- 技能等级（能力证明）
- 项目成果（业绩证明）
- 人际关系（软实力）

**晋升判定**：
- 综合评分 = 技能×0.4 + 业绩×0.4 + 人际×0.2
- 评分达标即可晋升
- 晋升失败可选择再试或跳槽

### 职场社交系统

**关系类型**：
- 导师：帮助你成长，提供指导
- 同事：日常协作，信息交换
- 领导：评估你，影响晋升
- 跨部门：拓展人脉，获取资源

**好感度影响**：
- 高好感度：获得帮助、信息、机会
- 低好感度：被孤立、信息闭塞、机会减少

### 长期目标系统

**目标类型**：
- 职业目标：3年内成为技术专家
- 财务目标：5年内存款50万
- 生活目标：工作生活平衡

**目标进度**：
- 可视化进度条
- 阶段性奖励
- 目标调整机制

## 随机事件池（规划）

### 工作相关

1. **项目紧急**：需要加班赶工
2. **技术难题**：遇到难以解决的问题
3. **同事求助**：帮助同事解决问题
4. **领导批评**：工作被批评，需要调整
5. **获得认可**：工作成果被表扬

### 社交相关

1. **团建活动**：参与团队建设
2. **同事聚餐**：建立职场人脉
3. **跨部门合作**：认识其他部门同事
4. **职场八卦**：处理办公室政治
5. **导师指导**：获得职业发展建议

### 个人成长

1. **技能瓶颈**：学习遇到困难
2. **自我提升**：业余时间学习
3. **职业规划**：思考未来发展
4. **工作生活平衡**：处理工作与生活
5. **健康问题**：过度劳累影响健康

## 结局分支（规划）

### 程序员路线结局

| 结局 | 触发条件 | 描述 |
|------|----------|------|
| 技术专家 | 晋升成功+技能等级高 | 成为团队技术骨干 |
| 稳步成长 | 晋升一般 | 按部就班发展 |
| 职场瓶颈 | 晋升失败+发展受阻 | 遇到职业瓶颈 |

### 运营产品路线结局

| 结局 | 触发条件 | 描述 |
|------|----------|------|
| 产品专家 | 晋升成功+业绩突出 | 成为产品负责人 |
| 运营骨干 | 晋升一般 | 在运营领域深耕 |
| 职场瓶颈 | 晋升失败 | 发展受限 |

### 稳扎稳打路线结局

| 结局 | 触发条件 | 描述 |
|------|----------|------|
| 成功融入 | 适应良好+晋升 | 完全适应新工作 |
| 平稳发展 | 适应一般 | 稳定发展 |
| 重新考虑 | 适应困难 | 考虑是否适合 |

### 延迟选择路线结局

| 结局 | 触发条件 | 描述 |
|------|----------|------|
| 后来居上 | 快速成长 | 追上同龄人 |
| 平淡稳定 | 发展一般 | 稳定的工作生活 |
| 继续迷茫 | 发展受阻 | 仍然找不到方向 |

## 数值设计要点

### 节奏控制

- 前15天：适应期，学习压力大
- 16-45天：成长期，工作逐渐上手
- 46-75天：挑战期，压力达到高峰
- 76-90天：总结期，回顾与展望

### 数值压力

- 精力：工作消耗大，需要平衡工作与生活
- 心情：职场压力影响大，需要调节
- 健康：久坐、熬夜、饮食不规律
- 存款：收入稳定，但消费也增加

### 长期规划

- 本章数值变化应该为后续章节做铺垫
- 晋升结果影响第四章的起点
- 人际关系影响后续发展
