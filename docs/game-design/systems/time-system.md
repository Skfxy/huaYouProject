# 时间推进系统

> 《话游》时间系统：天数、时段、日期事件

---

## 时间结构

### 基础单位

```
1天 = 4个时段
1周 = 7天
1月 = 4周（约30天）
1章 = 20-30天
```

### 时段划分

| 时段 | 时间范围 | 可用行为 |
|------|----------|----------|
| 早晨 | 6:00-12:00 | 学习、工作、早餐 |
| 下午 | 12:00-18:00 | 学习、工作、社交 |
| 晚上 | 18:00-24:00 | 学习、社交、娱乐 |
| 深夜 | 0:00-6:00 | 仅限熬夜行为 |

### 时间消耗规则

| 行为 | 消耗时段 | 说明 |
|------|----------|------|
| 学习1小时 | 1时段 | 最小单位1小时 |
| 工作1天 | 2时段 | 上午+下午 |
| 睡眠8小时 | 1时段 | 恢复精力 |
| 社交活动 | 1时段 | 聚餐、聊天 |
| 娱乐活动 | 1时段 | 游戏、刷视频 |

---

## 时间推进方式

### 主动推进

玩家选择行为后，时间自动推进：

```
选择行为 → 消耗时段 → 执行行为 → 数值变化 → 推进到下一时段
```

### 被动推进

特定事件触发时间跳跃：

```
触发条件 → 剧情事件 → 时间跳跃 → 新时段开始
```

### 时间跳跃示例

- 睡眠：直接跳到第二天早晨
- 强制休息：跳过半天或一天
- 剧情推进：根据事件跳过若干天

## 时间管理策略

### 精力分配建议

```
早晨（精力充沛）：学习高难度内容
下午（精力正常）：工作/学习
晚上（精力下降）：社交/轻松学习
深夜（精力耗尽）：避免熬夜，强制休息
```

### 时间价值评估

| 时段 | 价值系数 | 说明 |
|------|----------|------|
| 早晨 | 1.5x | 精力充沛，学习效率高 |
| 下午 | 1.0x | 正常效率 |
| 晚上 | 0.8x | 精力下降，适合轻松内容 |
| 深夜 | 0.5x | 效率低，且有健康惩罚 |

### 时间陷阱

- **熬夜陷阱**：深夜学习看似多时间，实际效率低且有惩罚
- **社交陷阱**：过度社交浪费时间，影响学习进度
- **娱乐陷阱**：刷视频时间过得快，但没有实质收益

## 时间与主线关联

### 程序员路线时间压力

```
第1-7天：学习基础语法
第8-14天：学习框架，做项目
第15-20天：准备面试，投递简历
关键节点：编程等级需达到2才能通过面试
```

### 运营产品路线时间压力

```
第1-7天：了解运营基础
第8-14天：学习文案写作
第15-20天：准备面试，投递简历
关键节点：运营等级需达到1才能通过面试
```

### 延迟选择路线时间压力

```
第1-10天：享受大学时光
第11-15天：开始焦虑，考虑未来
第16-20天：仓促准备，结果不确定
关键节点：心情>60才能触发好结局
```

## 时间系统参数

### 可调参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 每章天数 | 20天 | 第一章长度 |
| 每天时段数 | 4 | 早晨/下午/晚上/深夜 |
| 睡眠恢复量 | 40 | 精力恢复值 |
| 熬夜惩罚 | -15 | 精力额外消耗 |
| 学习效率衰减 | 50% | 连续学习2小时后 |

### 调参建议

- 增加每章天数：节奏更慢，更轻松
- 减少每天时段数：节奏更快，更紧张
- 提高睡眠恢复量：容错率更高
- 降低熬夜惩罚：鼓励熬夜策略
