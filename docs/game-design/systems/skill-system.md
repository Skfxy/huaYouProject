## 技能概览

### 第一章可用技能

| 技能 | 类型 | 等级范围 | 作用 |
|------|------|----------|------|
| 编程入门 | 专业技能 | 0-5 | 影响程序员路线面试通过率 |
| 运营基础 | 专业技能 | 0-5 | 影响运营路线面试通过率 |
| 社交能力 | 隐藏属性 | 0-10 | 影响人脉事件触发概率 |

### 技能分类

```
技能系统
├── 专业技能
│   ├── 编程入门 → 编程进阶 → 全栈开发
│   ├── 运营基础 → 内容运营 → 增长运营
│   └── 产品思维 → 产品设计 → 产品战略
├── 通用技能
│   ├── 社交能力 → 人脉拓展 → 资源整合
│   ├── 时间管理 → 效率提升 → 自律达人
│   └── 抗压能力 → 心态稳定 → 逆商高手
└── 隐藏技能
    ├── 好运光环（好感度解锁）
    ├── 直觉敏锐（特定事件解锁）
    └── 贵人相助（人脉系统解锁）
```

## 技能升级机制

### 经验获取方式

| 行为 | 经验值 | 条件 |
|------|--------|------|
| 学习1小时 | +20 | 精力>30 |
| 完成项目 | +50 | 实践类任务 |
| 获得好评 | +30 | 面试/作品被认可 |
| 教导他人 | +25 | 社交活动 |
| 深度思考 | +15 | 安静环境 |

### 升级所需经验

| 等级 | 所需经验 | 累计经验 | 效果加成 |
|------|----------|----------|----------|
| 0→1 | 100 | 100 | +10% |
| 1→2 | 200 | 300 | +20% |
| 2→3 | 350 | 650 | +35% |
| 3→4 | 500 | 1150 | +50% |
| 4→5 | 700 | 1850 | +75% |

### 经验衰减机制

- 同一技能连续学习超过2小时，经验获取-50%
- 精力<30时，经验获取-50%
- 心情<40时，经验获取-30%
- 建议：多种技能交替学习，保持效率

## 编程技能树

### 编程入门（等级0-5）

**解锁条件**：选择程序员路线

**等级效果**：

| 等级 | 能力 | 解锁内容 |
|------|------|----------|
| 0 | 了解概念 | 能看懂代码是什么 |
| 1 | 基础语法 | 能写简单程序，通过小公司面试 |
| 2 | 基础应用 | 能独立完成小项目，通过中等公司面试 |
| 3 | 进阶能力 | 能参与团队项目，通过大厂一面 |
| 4 | 熟练掌握 | 能独立负责模块，通过大厂二面 |
| 5 | 精通 | 能设计架构，通过大厂终面 |

**学习路径**：

```
等级0：了解HTML/CSS/JS概念
  ↓
等级1：学习JavaScript基础，能写TODO应用
  ↓
等级2：学习Vue/React框架，能写小型SPA
  ↓
等级3：学习Node.js，能写全栈项目
  ↓
等级4：学习系统设计，能做技术方案
  ↓
等级5：学习架构设计，能带团队
```

### 编程进阶（等级0-5，第二章解锁）

**解锁条件**：编程入门达到等级3

**分支方向**：

```
├── 前端方向
│   ├── Vue.js专家
│   ├── React专家
│   └── 小程序开发
├── 后端方向
│   ├── Node.js专家
│   ├── Java专家
│   └── Python专家
└── 全栈方向
    ├── 前后端通吃
    └── 架构师路线
```

## 社交能力（隐藏属性）

### 机制说明

- 不显示具体等级，通过事件结果体现
- 影响人脉事件触发概率

### 提升方式

| 行为 | 效果 |
|------|------|
| 主动搭话 | 社交+1 |
| 帮助他人 | 社交+2 |
| 组织活动 | 社交+3 |
| 拒绝社交 | 社交-1 |
| 说错话 | 社交-2 |

### 解锁内容

| 社交等级 | 解锁内容 |
|----------|----------|
| 低 | 基础社交事件 |
| 中 | 人脉事件触发，获得学长联系方式 |
| 高 | 隐藏剧情，贵人相助事件 |

## 技能与主线关联

### 程序员路线

- 核心技能：编程入门 → 编程进阶
- 辅助技能：社交能力（内推机会）
- 关键节点：编程等级3可通过大厂一面

### 运营产品路线

- 核心技能：运营基础 → 增长运营
- 辅助技能：社交能力（用户洞察）
- 关键节点：运营等级2可通过中等公司面试

### 稳扎稳打路线

- 核心技能：编程入门（业余学习）
- 辅助技能：时间管理（平衡工作学习）
- 关键节点：编程等级2可成功转行

### 延迟选择路线

- 核心技能：社交能力（人脉积累）
- 辅助技能：任意（后期压力大）
- 关键节点：社交等级3可获得意外机遇

## 数值平衡参考

### 学习效率曲线

```
精力充沛（>80）：效率 150%
精力正常（30-80）：效率 100%
精力不足（<30）：效率 50%
```

### 技能成长建议

- 前期：重点提升一个专业技能，快速达到等级2
- 中期：专业技能+社交能力并行发展
- 后期：根据主线选择深度发展方向
- 避免：平均分配经验，导致没有核心竞争力
